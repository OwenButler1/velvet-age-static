# Velvet Age Static

A static, compliance-first landing page template for a lawful age-restricted adult members website.

## Pages

- `index.html` — homepage with age gate, collection cards, pricing, and compliance sections
- `terms.html` — placeholder terms page
- `privacy.html` — placeholder privacy page
- `report.html` — placeholder reporting page
- `styles.css` — full responsive styling
- `script.js` — local browser age-gate confirmation

## Important

The built-in age gate is only a UX placeholder. Before launching an adult-content website, integrate a legally appropriate strong age-assurance provider, performer verification, consent records, reporting workflows, and lawful payment processing.
